WDD 131 - Dynamic Web Fundamentals

This course is designed to teach students the fundamentals of dynamic web development. Students will learn how to create dynamic web pages using HTML, CSS, and JavaScript. Students will also learn how to use web development tools such as Git, GitHub, and Visual Studio Code. Students will learn how to create responsive web pages that work on desktops, tablets, and mobile devices. Students will also learn how to create web pages that interact with databases and web services. Students will learn how to create web pages that use AJAX to update content without reloading the page. Students will learn how to create web pages that use JavaScript to create interactive user interfaces. Students will learn how to create web pages that use JavaScript to validate user input. Students will learn how to create web pages that use JavaScript to create animations and transitions. Students will learn how to create web pages that use JavaScript to create games and simulations. Students will learn how to create web pages that use JavaScript to create charts and graphs. Students will learn how to create web pages that use JavaScript to create maps and geolocation services. Students will learn how to create web pages that use JavaScript to create web components. Students will learn how to create web pages that use JavaScript to create web workers. Students will learn how to create web pages that use JavaScript to create service workers. Students will learn how to create web pages that use JavaScript to create progressive web apps. Students will learn how to create web pages that use JavaScript to create web assembly modules. Students will learn how to create web pages that use JavaScript to create web sockets. Students will learn how to create web pages that use JavaScript to create webRTC services. Students will learn how to create web pages that use JavaScript to create webGL graphics. Students will learn how to create web pages that use JavaScript to create webVR experiences. Students will learn how to create web pages that use JavaScript to create webAR experiences. Students will learn how to create web pages that use JavaScript to create webXR experiences. Students will learn how to create web pages that use JavaScript to create web animations. Students will learn how to create web pages that use JavaScript to create web transitions. Students will learn how to create web pages that use JavaScript to create web games.
